# Predictive Model for Prepaid to Postpaid Migration

## Objective:
  - To increase migration from Prepaid to Postpaid

## Impact on Business
  - Increase in Postpaid Revenue and subscriber base
  
## Target Definition:
  - Target = 1
    - Non Migrator
  - Target = 0
    - Migrator

## Dependent Variables:
  - Usages (Voice, SMS, Data, VAS)
  - Demographic
